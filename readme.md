### Empirical Analysis of Phase Transitions in Transformer Models Using Mechanistic Interpretability


## Abstract

This student–faculty collaboration project documents, in mathematical and conceptual detail, phase transitions in transformer models. Our dataset consists of Fibonacci sequences modulo (n). Late generalization on algorithmic tasks, such as Fibonacci modulo (n), can be observed in these types of models.

<hr />

We consider sequences generated by the Fibonachi modulo n.

xt+1 = xt + xt−1 (mod n).

Each token belongs to the voculabulary:

Zn = {0, 1, . . . , n − 1}.
